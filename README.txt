README – Project Execution Instructions

To run the project's Jupyter Notebooks, Python 3 is required, along with the following libraries:

- pandas
- scipy
- openpyxl
- networkx
- matplotlib

Install the libraries using pip:
pip install pandas scipy openpyxl networkx matplotlib

Alternatively, in an Anaconda environment:
conda install pandas scipy openpyxl networkx matplotlib

The remaining libraries used in the project are included in Python's standard library and do not require additional installation.

Instruction for the notebook "Queueing_Systems.ipynb":
In this notebook, the user can test a different number of servers/employees (c) by entering the value when requested by the program.
To run each example, press Shift + Enter in the corresponding cell.
